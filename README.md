# Flappy Bird - Edisi Pygame
> Remodel dari game klasik dengan animasi dan efek ledakan!

## Instalasi
```bash
git clone https://github.com/Jodi-Therson/flappy-bird.git
cd flappy-bird
pip install requirements.txt
python main.py
```
